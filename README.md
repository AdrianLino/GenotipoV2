#  Tabajo Genotipo
 Creación de un módulo administrable, donde se puedan visualizar, agregar, 
actualizar y eliminar artículos. 

## Tecnologias:
- codeigniter V4
- Tailwind
- JavaScript
- MySQL
- PHP
